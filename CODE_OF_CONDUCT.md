# Code of Conduct

이 프로젝트는 [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) 을 준수합니다. 본문은 위 URL 을 참조하세요. 한국어 번역은 [contributor-covenant.org/translations/ko/](https://www.contributor-covenant.org/ko/version/2/1/code_of_conduct/) 에 있습니다.

## 적용 범위

GitHub Issues, Pull Requests, 코드 주석, 이 저장소가 호스팅하는 문서·블로그·데모 — 모든 프로젝트 공간에 적용됩니다.

## 신고

위반 사례는 메인테이너에게 직접 연락:

- GitHub: [@HardcoreMonk](https://github.com/HardcoreMonk)

신고는 비공개로 처리됩니다.

## 시행

메인테이너는 Contributor Covenant 의 [시행 가이드라인](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines) (수정·경고·일시 차단·영구 차단 4단계) 을 따릅니다.
